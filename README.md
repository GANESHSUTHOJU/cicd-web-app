# 🚀 CI/CD Pipeline for Flask Web Application

## 📌 Project Overview

This project demonstrates a CI/CD pipeline using GitHub Actions and Docker.  
A Flask web application is containerized and automatically built on every code push.

---

## 🌍 Live Demo

https://cicd-web-app-bw0g.onrender.com

## 🛠️ Technologies Used

- Python (Flask)
- Docker
- GitHub Actions (CI/CD)

---

## ⚙️ Features

- Simple Flask web application
- Docker containerization
- Automated CI/CD pipeline
- Health check endpoint
